\subsection{Deep Reinforcement Learning Implementation}

\subsubsection{Deep Q-Network (DQN) Architecture}

The agent's decision-making policy is approximated using a Deep Q-Network
(DQN)~\cite{mnih2015humanlevel}. We employ a Multi-Layer Perceptron (MLP) architecture to
map the high-dimensional state vector
\(\mathbf{S}_t\) to Q-values for the five discrete actions.

The network is a 3-layer MLP with hidden dimensions [512, 512, 256] and ReLU activations.
The input dimension is $4(3+2N)$ (frame-stacked state) and the output is 5 Q-values,
one per action in $\mathcal{A}$.

The network weights \(\theta\) are optimised using the Adam optimiser to minimise the temporal
difference error between the predicted Q-values and the target Q-values derived from the
Bellman equation. To ensure training stability, a separate target network \(\hat{Q}\) with
parameters \(\theta^{-}\) is maintained and updated periodically (every \(C\) steps).

\subsubsection{Custom Gymnasium Environment Design}

The simulation environment is implemented as a custom Gymnasium
environment~\cite{towers2024gymnasium}, providing a standardised interface that enables
seamless integration with the Stable-Baselines3~\cite{raffin2021stable} DQN
implementation. The environment incorporates a probabilistic link-layer model to move beyond
binary `in-range' abstractions. Each sensor's transmission success is determined by the
product of its protocol-mandated duty cycle and a sigmoid-mapped signal-to-noise ratio (SNR).
This mapping calculates the instantaneous success probability \(P_{\text{overall}}\) by
comparing the RSSI (derived from the Two-Ray Ground Reflection model) against the noise floor
and the specific SNR requirements of the current Spreading Factor. By setting a transmission
threshold (e.g., \(P_{\text{overall}} > 0.5\)), the environment forces the DQN agent to
navigate the trade-off between proximity-induced signal strength and the stochastic
availability of sensor nodes. This in turn allows for the creation and editing of link
quality.

\subsubsection{Training Convergence}

The DQN was trained across 16 sequential curriculum stages. Reward converges within each
stage (EMA curve rises and plateaus) and later stages converge to progressively higher
rewards ($3$--$5\times10^6$), confirming that the curriculum successfully bootstraps the
agent from easy to hard conditions. The TD loss follows the same spike-then-converge
profile in every stage with no divergence across transitions. Full convergence curves
are shown in Appendix~\ref{app:training}.
