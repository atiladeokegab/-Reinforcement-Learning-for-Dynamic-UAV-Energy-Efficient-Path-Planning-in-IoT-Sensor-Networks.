\subsection{Aims and objectives}

The primary aim of this project is to develop a protocol-aware Deep Reinforcement Learning
(DRL) framework capable of optimising UAV flight trajectories for energy-efficient data
collection in dense, unsynchronised LoRaWAN networks. The goal is to demonstrate that an
intelligent agent can resolve the ``Spreading Factor (SF) Monoculture'' bottleneck by using
its physical position as a direct control input for the radio protocol.

Specifically, the project will train a Deep Q-Network (DQN) agent within a high-fidelity
simulation to learn a non-obvious, network-aware policy that maximises data throughput while
minimising energy consumption.

The main objectives of the project are summarised below:

\begin{itemize}
    \item To develop a high-fidelity simulation environment compatible with the Gymnasium
    interface, incorporating rigorous telecommunications physics including the Two-Ray Ground
    Reflection path loss model, log-normal shadowing, and asynchronous duty cycles. To the
    author's knowledge, this is the first open framework to model the complete causal chain
    from UAV position through RSSI, EMA-ADR, and Capture Effect to packet delivery within a
    single simulation step.

    \item To model the specific constraints of the LoRaWAN protocol, particularly the
    Adaptive Data Rate (ADR) latency, to replicate the causal link between UAV positioning,
    RSSI, and Spreading Factor allocation.

    \item To formulate the UAV data collection problem as a Markov Decision Process (MDP),
    designing a multi-objective reward function that balances the competing goals of energy
    conservation, data freshness (AoI), and collection volume.

    \item To train a Deep Q-Network (DQN) agent to learn an emergent flight policy that
    autonomously navigates the ``Starvation vs.\ Throughput'' trade-off without human
    intervention.

    \item To evaluate the performance of the protocol-aware DRL agent by comparing it against
    two greedy path-planning baselines --- Nearest-Sensor Greedy and SF-Aware Max-Throughput
    Greedy --- in terms of energy efficiency, fairness (Jain's Index), and total bytes
    delivered, using rigorous multi-seed statistical testing (Welch's $t$-test, Cohen's $d$).

    \item To assess the trade-offs between computational complexity, policy robustness, and
    real-world feasibility, identifying the ``Sim-to-Real'' gap as a key area for future
    research.
\end{itemize}

Collectively, these objectives demonstrate that a protocol-aware agent can generate
energy-efficient trajectories satisfying both network fairness and throughput requirements
without relying on pre-specified routing heuristics.
